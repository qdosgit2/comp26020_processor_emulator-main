8// The classes in this file describe the instructions supported by our emulator.
9// Our instructions are simple:
10// - Arithmetic, logical, and memory instructions operate on the accumulator and
11// a memory address
12// - Only one arithemetic instruction is implemented: ADD. For subtraction, you
13// need to add with the two's complement of the second value. To invert the
14// second argument, you need to XOR with 0xff.
15// - Only three logical instructions are implemented: AND, ORR, XOR. Other
16// operations are composed from those three
17// - Only one conditional branch is implemented: JNE (Jump if Not Equal). This
18// is generally not enough for real algorithms, but supporting more branches
19// would make this codebase unnecessarily long
20// -----------------------------------------------------------------------------
